# Usage

Start development servers with

```
npm start
```
This will use concurrently to run <code>npm start</code> under client and server directories.
## Folder Structure
```
trivianation/
    client/
        src/
            index.js
    server/
        src/
            index.js
        public/
            index.html
```

For the project to build, **the above files must exist**.
